## Flujo de verificación (en construcción)

1. Login con Google o email/password.
2. Pega un texto, una URL, o sube un video.
3. El sistema procesa en vivo (ves los eventos: claims extraídos, fuentes encontradas, ...).
4. Recibes el reporte estructurado con:
   - Cada claim identificado
   - Evidencia que lo apoya o contradice, con citas y enlaces
   - Score de confianza desglosado por factor
   - Señales de manipulación detectadas en el input original
   - Brechas de evidencia
